# xrobot - XR-E Pro Robot Development Kit

Private robot development kit for XR-E Pro humanoid robot.

## Structure
- docker/: ROS1 Docker environment
- src/ros1/: ROS1 tools and nodes
- src/lerobot/: LeRobot imitation learning
- hardware/xr1/: XR-1 robot configuration
- training/: Training scripts
- deployment/: Deployment tools
